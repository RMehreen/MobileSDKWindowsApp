Capture Picture

WinForm Capture Picture sample program

This sample illustrates how to capture picture from the connected device, and how to subscribe to the PictureCaptured event.

For Capture Picture to work, please ensure that Client Drive Mapping (CDM) is enabled, and full access is granted for the SD card on the connected device.

Source

winform/capturepicture/Form1.cs

The cmp referenced in the code snippets below is an instance of the CitrixMobile class.

CapturePicture

The following is an example of how to capture picture from the device.

options is a CMP_CAPTURE_PICTURE_OPTIONS structure used to specify the desired options for the captured picture, this structure must be initialised before calling CMPCapturePicture.

uniqueId is an unique identifier returned from the CapturePicture call. This identifier will be referenced in a subsequent PictureCaptured event when the picture is acquired.

                    // Request a picture capture.
                    int rc = cmp.CapturePicture(ref options, out captureId);

Event subscription

The CitrixMobile class exposes an event called CMPCapturePicture.

When the CMPCapturePicture request is fulfilled by the connected device, the application will be notified via a PictureCaptured event when the picture is ready.

The following is an example of how to subscribe to the PictureCaptured event.

                    // Register for PictureCaptured event.
                    cmp.PictureCaptured += new ICMPEvents_PictureCapturedEventHandler(cmp_PictureCaptured);
                    cmp.PictureCaptured += new ICMPEvents_PictureCapturedEventHandler(cmp_PictureCaptured);

Event handler

The following is an implementation of the PictureCaptured event handler.

uniqueId is an unique identifier returned in a preceding CMPCapturePicture request.
Filename is the location of the picture, this is provided as an UNC path which corresponds to the Client Drive Mapping for the connected device.

        // <summary>
        // PictureCaptured event handler.
        // </summary>
        // <param name="rc">Capture result.</param>
        // <param name="uniqueId">Capture identifier.</param>
        // <param name="pictureMetadata">Capture metadata filename.</param>
        // <param name="filename">Capture filename.</param>
        // <param name="thumbnail">Thumbnail filename.</param>
        // <param name="PictureSize">Capture size.</param>
        // <param name="Quality">Capture quality.</param>
        // <param name="CameraSelection">Camera selected.</param>
        // <param name="Encoding">Capture encoding.</param>
        // <param name="DesiredWidth">Capture width.</param>
        // <param name="DesiredHeight">Capture height.</param>
        // <param name="DurationLimit">Capture duration limit.</param>
        // <param name="ThumbnailType">Thumbnail type.</param>
        // <param name="ThumbnailWidth"Capture width.</param>
        // <param name="ThumbnailHeight">Capture height.</param>
        void cmp_PictureCaptured(CMPRESULT rc, CMP_UNIQUE_LONG_ID uniqueId, string pictureMetadata, string filename, string thumbnail, 
            CMPRESULT PictureSize, CMP_CAPTURE_QUALITY Quality, CMP_CAPTURE_CAMERA CameraSelection, CMP_IMAGE_FORMAT Encoding, CMPRESULT DesiredWidth, 
            CMPRESULT DesiredHeight, CMP_THUMBNAIL_TYPE ThumbnailType, short ThumbnailWidth, short ThumbnailHeight)
        {
            // only process capture if it is one of ours
            if (uniqueId == captureId)
            {
                if (CMP_SUCCESS(rc))
                {
                    lock (stateLock)
                    {
                        processingCapture = true;
                    }

                    UpdateStatus(string.Format("Capture received: {0}", filename));

                    //Save the capture to the local temporary directory (keeping the same file name and extension).
                    string newLocation = System.IO.Path.GetTempPath() + System.IO.Path.GetFileName(filename);

                    UpdateStatus(string.Format("Saving capture to: {0}", newLocation));

                    File.Copy(filename, newLocation, true);

                    // Display the capture. Note that this is invoked in the UI thread.
                    ShowPicture(newLocation);
                    UpdateStatus(string.Format("Capture saved: {0}", newLocation));

                    lock (stateLock)
                    {
                        processingCapture = false;
                    }
                }
            }
        }
