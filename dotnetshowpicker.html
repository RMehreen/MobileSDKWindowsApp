Show Picker

C# Show Picker sample program

This sample illustrates how to show the Picker control of the connected device, and how to subscribe to the PickerControlStateChanged event.

Source

csharp/showpicker/Program.cs

The cmp referenced in the code snippets below is an instance of the CitrixMobile class.

ShowPicker

The following is an example of how to show the picker control of the device.

pickerId is an application specific unique identifier. This identifier will be referenced in a subsequent PickerControlStateChanged event.

            // Show the picker control.
            rc = cmp.ShowPicker(PickerId, ref rect, 0, ref pickerText, pickerTitle);

Event subscription

The CitrixMobile class exposes an event called PickerControlStateChanged.

When the ShowPicker request is fulfilled by the connected device, the application will be notified via a PickerControlStateChanged event.

The following is an example of how to subscribe to the PickerControlStateChanged event.

            // Subscribed PickerControlStateChanged event.
            cmp.PickerControlStateChanged += new ICMPEvents_PickerControlStateChangedEventHandler(cmp_PickerControlStateChanged);

Event handler

The following is an implementation of the PickerControlStateChanged event handler.

PickerId is an application specific unique identifier specified in a preceding CitrixMobility.IPicker.ShowPicker request.

        // <summary>
        // PickerControlStateChanged event handler.
        // </summary>
        // <param name="pickerId">Picker identifier.</param>
        // <param name="pickerStateFlags">Picker state flags.</param>
        // <param name="rc">Return code.</param>
        // <param name="selectedIndex">Selected index.</param>
        static void cmp_PickerControlStateChanged(
            CMP_UNIQUE_ID pickerId,
            Int16 pickerStateFlags,
            CMPRESULT rc,
            Int16 selectedIndex)
        {
            CMP_PICKER_CONTROL_STATE pickerState = (CMP_PICKER_CONTROL_STATE)pickerStateFlags;

            Console.WriteLine("Picker control state changed pickerId({0:X}) flags({1:X}) rc({2:X} selectedIndex({3})",
                              pickerId, pickerStateFlags, rc, selectedIndex);

            // only proceed if picker is one of ours
            if (pickerId == PickerId)
            {
                if (IsPickerVisible(pickerState))
                {
                    Console.WriteLine("Picker is visible");
                }

                if (IsPickerSelected(pickerState))
                {
                    Console.WriteLine("Picker item has been selected.  Index({0}) Item({1})",
                                      selectedIndex, pickerText[selectedIndex]);
                }

                if (IsPickerAborted(pickerState))
                {
                    Console.WriteLine("Picker control was aborted");
                }
            }
        }
