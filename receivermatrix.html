Receiver Features Matrix

Support for specific Citrix Mobile SDK for Windows Apps features are platform dependent. Availability of some features are also dependent on the current Receiver implementation on each of the supported platforms.

Listed below is a summary of Citrix Mobile SDK for Windows Apps features supported on the respective Receiver platforms. In order to ensure correct functionality of applications making use of the Citrix Mobile SDK for Windows Apps, developers should consult the table below and cater for each of the Receiver platform as appropriate.
