Send SMS

C++ Send SMS sample program

This sample illustrates how to call the built-in SMS application of the connected device, and how to subscribe to the SMSStarted event.

Source

native/sendsms/sendsms.cpp

The hCMP referenced in the code snippets below is the handler for the CitrixMobile class.

SendSMS

The following is an example of how to request an SMS to be made by the device.

UNIQUE_SMS_ID is an application specific unique identifier. This identifier will be referenced in a subsequent SMSStarted event.

    // Start the SMS process by popping up the SMS screen with the phone number already populated
    rc = CMPSendSMS(hCMP, phoneNumber, SMSId, SMSText);

Event subscription

The CitrixMobile class exposes an event called SMSStarted.

When the CMPSendSMS request is fulfilled by the connected device, the application will be notified via a SMSStarted event.

The following is an example of how to subscribe to the SMSStarted event.

    // Subscribe to SMSStarted event.
    rc = CMPRegisterForEvent(hCMP, CMP_EVENT_SMS_STARTED, (CMP_EVENT_CALLBACK)SMSStarted);

Event handler

The following is an implementation of the SMSStarted event handler.

UNIQUE_SMS_ID is an application specific unique identifier specified in a preceding SMSStarted request.

// <summary>
// SMSStarted event handler.
// </summary>
// <param name="hCMP">CMP handler.</param>
// <param name="rc">Return code.</param>
// <param name="SMSId">SMS identifier.</param>
void CMPAPI SMSStarted(HANDLE hCMP, CMPRESULT rc, CMP_UNIQUE_ID SMSId)
{
    // only proceed if the sms is one of ours
    if(SMSId == UNIQUE_SMS_ID)
    {
        printf("SMSStarted hCMP(%p) rc(0x%X) SMSId(%X)\n", hCMP, rc, SMSId);
    }
}
