Supported Orientations

C# Supported Orientations sample program

This sample illustrates how to get and set the supported orientations information of the connected device, and how to subscribe to the SupportedOrientationsChanged event.

Source

csharp/supportedorientations/Program.cs

The cmp referenced in the code snippets below is an instance of the CitrixMobile class.

GetSupportedOrientations

The following is an example of how to get the supported orientations of the device.

                    // Gets the current supported orientations of the device
                    rc = cmp.GetSupportedOrientations(out supportedOrientations);

SetSupportedOrientations

The following is an example of how to set the supported orientations of the device.

                    // Sets the supported orientations to Landscape Only
                    rc = cmp.SetSupportedOrientations(CMP_SUPPORTED_ORIENTATIONS.CMP_SO_LANDSCAPE);

Event subscription

The CitrixMobile class exposes an event called SupportedOrientationsChanged.

When a change in the supported orientations of the connected device is detected, the application will be notified via a SupportedOrientationsChanged event.

The following is an example of how to subscribe to the SupportedOrientationsChanged event.

            // Subscribe to SupportedOrientations changed event.
            cmp.SupportedOrientationsChanged += new ICMPEvents_SupportedOrientationsChangedEventHandler(cmp_SupportedOrientationsChanged);

Event handler

The following is an implementation of the SupportedOrientationsChanged event handler.
