Feature Capabilities

WinForm Capabilities sample program

The idea of supported capabilities stems from the concept of negotiated feature sets between client and server. The resulting set of supported capabilities can be determined using SDK API.

This sample program retrieves the different capabilities available. Review the Feature Capabilities for more detail on the concept.

It is important to remember that every request for a capability requires a capability Id CMP_CAP_ID and a key Id for the requested capability.

The type of the capability determines the value format. At this time, capabilities only include numeric values.

Note that these information are initialised upon session connection, and will remain unchanged for the duration of the session. For example, the display width and height reflect the resolution at the time of the session connection, and these values will not change regardless of orientation of the device during this session.

winform/capabilities/Form1.cs

The cmp referenced in the code snippets below is an instance of the CitrixMobile class.

The following are examples of how to retrieve various device capabilities:

Display Capabilities

        // Get the display DPI of the connected device.
        rc = cmp.GetCapabilityInt32(CMP_CAP_ID.CAPID_DISPLAY_INFO, (short)CMP_CAP_DISPLAY_KEY_ID.CAP_DISPLAY_PIXELS_PER_INCH, out iValue);

        // Get the display width of the connected device.
        rc = cmp.GetCapabilityInt32(CMP_CAP_ID.CAPID_DISPLAY_INFO, (short)CMP_CAP_DISPLAY_KEY_ID.CAP_DISPLAY_WIDTH, out iValue);

Keyboard Capabilities

        // Determine whether the ability to show/hide on-screen keyboard is enabled on the connected device.
        rc = cmp.GetCapabilityBool(CMP_CAP_ID.CAPID_INPUT, (short)CMP_CAP_INPUT_KEY_ID.CAP_INPUT_HIDE, out bValue);

        // Determine whether the standard on-screen keyboard is enabled on the connected device.
        rc = cmp.GetCapabilityBool(CMP_CAP_ID.CAPID_INPUT, (short)CMP_CAP_INPUT_KEY_ID.CAP_INPUT_STD_KYBD, out bValue);

Phone Capabilities

        // Determine whether sending SMS is enabled on the connected device.
        rc = cmp.GetCapabilityBool(CMP_CAP_ID.CAPID_SMS, (short)CMP_CAP_SMS_KEY_ID.CAP_SMS_ENABLE, out bValue);

        // Determine whether phone dialing is enabled on the connected device.
        rc = cmp.GetCapabilityBool(CMP_CAP_ID.CAPID_PHONE_CALL, (short)CMP_CAP_PHONE_CALL_KEY_ID.CAP_PHONE_CALL_ENABLE, out bValue);
