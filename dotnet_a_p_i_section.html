C# COM Interop API

In .NET C#, the Citrix Mobile SDK for Windows Apps is accessed by referencing the Interop.CitrixMobility.DLL in the C# project. Once referenced, it is then possible to create a CitrixMobility.CitrixMobile object to interact with the mobile device. The CitrixMobile object is actually a COM object using the COM Interop feature of .NET.

Classes

  • CitrixMobility.CitrixMobile The default interface for CitrixMobile is ICMPDispatch. It is not necessary to cast the CitrixMobile object to the different interfaces if the API is included in ICMPDispatch.
  • CitrixMobility.CitrixMobileClass Collection of all the interfaces available to the CitrixMobile object.
