Send SMS

C# Send SMS sample program

This sample illustrates how to call the built-in SMS application of the connected device, and how to subscribe to the SMSStarted event.

Source

csharp/sendsms/Program.cs

The cmp referenced in the code snippets below is an instance of the CitrixMobile class.

SendSMS

The following is an example of how to request an SMS to be made by the device.

phoneNumber is the designated number for the SMS to be sent.
SMSId is an application specific unique identifier. This identifier will be referenced in a subsequent SMSStarted event.
SMSText is the string content of the SMS.

            // Start the SMS process by popping up the SMS screen with the phone number already populated
            rc = cmp.SendSMS(phoneNumber, SMSId, SMSText);

Event subscription

The CitrixMobile class exposes an event called SMSStarted.

The following is an example of how to subscribe to the SMSStarted event. When the CMPSendSMS request is fulfilled by the connected device, the application will be notified via a SMSStarted event.

            // Subscribed SMSStarted event.
            cmp.SMSStarted += new ICMPEvents_SMSStartedEventHandler(cmp_SMSStarted);

Event handler

The following is an implementation of the SMSStarted event handler.

SMSId is an application specific unique identifier specified in a preceding SMSStarted request.

        // <summary>
        // SMSStarted event handler.
        // </summary>
        // <param name="rc">Return code.</param>
        // <param name="SMSId">SMS identifier.</param>
        static void cmp_SMSStarted(CMPRESULT rc, CMP_UNIQUE_ID smsId)
        {
            Console.WriteLine("SMS started rc({0:X}) SMSId({1:X})", rc, smsId);

            //only proceed if sms is one of ours
            if (SMSId == smsId)
            {
                // Do something...
            }
        }
