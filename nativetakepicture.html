Take Picture

C++ Take Picture sample program

This sample illustrates how to take a picture by calling the built-in camera application of the connected device, and how to subscribe to the PictureTaken event.

For Take Picture to work, please ensure that Client Drive Mapping (CDM) is enabled, and full access is granted for the SD card on the connected device.

Source

native/takepicture/takepicture.cpp

The hCMP referenced in the code snippets below is the handler for the CitrixMobile class.

TakePicture

The following is an example of how to request a picture to be taken by the device.

pictureId is an application specific unique identifier. This identifier will be referenced in a subsequent PictureTaken event when the picture is acquired.

    // request a picture to be taken
    rc = CMPTakePicture(hCMP, pictureId, imageFormat);

Event subscription

The CitrixMobile class exposes an event called PickerControlStateChanged.

When the CMPTakePicture request is fulfilled by the connected device, the application will be notified via a PictureTaken event when the picture is ready.

The following is an example of how to subscribe to the PictureTaken event.

    // Subscribed PictureTaken event.
    rc = CMPRegisterForEvent(hCMP, CMP_EVENT_CAMERA_PICTURE_TAKEN, (CMP_EVENT_CALLBACK)PictureTaken);

Event handler

The following is an implementation of the PictureTaken event handler.

pictureId is an application specific unique identifier specified in a preceding CMPTakePicture request. The location of the picture is provided via an UNC path, which corresponds to the client drive mapping for the connected device.

// <summary>
// PictureTaken event handler.
// </summary>
// <param name="hCMP">CMP handler.</param>
// <param name="result">Picture result.</param>
// <param name="pictureId">Picture identifier.</param>
// <param name="pictureFormat">Picture format.</param>
// <param name="pictureSize">Picture size.</param>
// <param name="filename">Picture filename.</param>
void CMPCALL PictureTaken(
        HANDLE hCMP, 
        CMPRESULT result, 
        CMP_UNIQUE_ID pictureId,
        CMP_IMAGE_FORMAT pictureFormat, 
        UINT32 pictureSize, 
        UTF8_STRING filename)
{
    printf("PictureTaken event\n");

    // Copy the relevant data and then signal the event
    g_PictureData.pictureResult = result;
    g_PictureData.pictureId     = pictureId;
    g_PictureData.pictureFormat = pictureFormat;
    g_PictureData.pictureSize   = pictureSize;
    g_PictureData.filename      = _strdup(filename);

    printf("Picture result(0x%X) id(0x%X) format(%u) size(%u) filename(%s)\n", 
            result, pictureId, pictureFormat, pictureSize, filename);

    SetEvent(hPictureEvent);
}
