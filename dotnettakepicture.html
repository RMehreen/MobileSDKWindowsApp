Take Picture

C# Take Picture sample program

This sample illustrates how to call the built-in camera application of the connected device, and how to subscribe to the PictureTaken event.

For Take Picture to work, please ensure that Client Drive Mapping (CDM) is enabled, and full access is granted for the SD card on the connected device.

Source

csharp/takepicture/Program.cs

The cmp referenced in the code snippets below is an instance of the CitrixMobile class.

TakePicture

The following is an example of how to request a picture to be taken by the device.

pictureId is an application specific unique identifier. This identifier will be referenced in a subsequent PictureTaken event when the picture is acquired.

            // Request a JPEG picture to be taken.
            rc = cmp.TakePicture(CMP_IMAGE_FORMAT.CMP_IMAGE_FORMAT_JPEG, PictureId);

Event subscription

The following is an example of how to subscribe to the PictureTaken event. When the CMPTakePicture request is fulfilled by the connected device, the application will be notified via a PictureTaken event when the picture is ready.

            // Subscribed PictureTaken event.
            cmp.PictureTaken += new ICMPEvents_PictureTakenEventHandler(cmp_PictureTaken);

Event handler

The following is an implementation of the PictureTaken event handler.

pictureId is an application specific unique identifier specified in a preceding CMPTakePicture request. The location of the picture is provided via an UNC path, which corresponds to the client drive mapping for the connected device.

        // <summary>
        // PictureTaken event handler.
        // </summary>
        // <param name="result">Picture result.</param>
        // <param name="pictureId">Picture identifier.</param>
        // <param name="pictureFormat">Picture format.</param>
        // <param name="pictureSize">Picture size.</param>
        // <param name="filename">Picture filename.</param>
        static void cmp_PictureTaken(
            CMPRESULT result,
            CMP_UNIQUE_ID pictureId,
            CMP_IMAGE_FORMAT pictureFormat,
            int pictureSize,
            string filename)
        {
            // only process picture if it is one of ours
            if (pictureId == PictureId)
            {
                Console.WriteLine("PictureTaken event\n");

                // Copy the relevant data and then signal the event
                pictureData.pictureResult   = result;
                pictureData.pictureId       = pictureId;
                pictureData.pictureFormat   = pictureFormat;
                pictureData.pictureSize     = pictureSize;
                pictureData.filename        = filename;

                Console.WriteLine("Picture result({0:X}) id({1:X}) format({2:X}) size({3:X}) filename(4)\n",
                                   result, pictureId, pictureFormat, pictureSize, filename);

                if (CMP_SUCCESS(pictureData.pictureResult))
                {
                    Console.WriteLine("Picture id({0:X}) format({1:X}) size({2:X}) filename({3})\n",
                                      pictureData.pictureId, pictureData.pictureFormat,
                                      pictureData.pictureSize, pictureData.filename);

                    // Saves the image as "img.jpg" in the "C:\photos" directory.
                    // Directory needs to be created beforehand.
                    string newLocation = "C:\\photos\\img.jpg";

                    Console.WriteLine("CMPGetPictureFilename mobileFilename({0}) newLocation({1})", filename, newLocation);

                    Console.WriteLine("Copying file..\n");

                    File.Copy(filename, newLocation, true);

                    Console.WriteLine("Copy succeeded\n");

                    pictureProcessed.Set();
                }
            }
        }
