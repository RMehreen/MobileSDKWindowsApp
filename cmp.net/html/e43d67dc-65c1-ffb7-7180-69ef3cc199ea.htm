ClientViewport

The client viewport defines the screen area on the client device that is available to view the server session. The resolution that is reported in the DisplayState class is the full screen resolution of the client device. However many Citrix receivers will not have access to this full resolution to display the session, because of toolbars. This viewport value takes these things into account, so it will return the client resolution minus any space taken up by toolbars or other onscreen elements, e.g. the onscreen keyboard.

If the client viewport was not available then this will be null.

ServerViewport

When viewing a remote session, the size of the session maybe bigger than the size of the client viewport. This means you cannot view the entire server session at once. Receivers provide a panning facility that allows users to move the viewport around to view the entire server session. This viewport value reports which section of the server session is visible inside the client viewport.

If the server viewport was not available then this will be null.

ZoomFactor
The server viewport zoom factor as a percentage, e.g. 100 means no zooming. When the value is null the zoom factor is not available. When viewport events occur that do not affect the zoom factor, this value will be null.
