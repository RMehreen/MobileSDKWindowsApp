VideoCaptureVideo Method Citrix Mobility Pack .NET SDK
Performs a synchronous operation to capture a video clip. Since this operation can take a long time to complete, calling this synchronous version of the API is not recommended. Instead you should use the async version BeginCaptureVideo(VideoCaptureOptions, TimeSpan, ActionIAsyncResult) so you are not blocked while the operation is in progress.

The options for the capture operation.
timeout
Type: SystemTimeSpan
The amount of time to wait for the user to capture an video clip before timing out the operation. You must supply a value greater than 0 and less than or equal to 60 minutes, or an exception will be thrown.

Return Value

Type: VideoClip
The captured video clip if the operation was successful.
