DisplayBeginSetViewportOrigin Method Citrix Mobility Pack .NET SDK

When the resolution of the HDX session is bigger than the size of the client's screen the user is able to pan the viewport around on their client. This function allows the hosted app on the server to programmatically alter this viewport position.

This function initiates an async request to set the viewport position. To check the result of the operation call EndSetViewportOrigin(IAsyncResult).

You can use CanSetViewport to check whether the current client supports setting the viewport position. If it does not support it, the operation will throw a NotSupportedException when this async operation is completed.
