Remarks
The scaling is updated from the client capabilities as soon as the API is initialised or a session reconnection occurs. Since the capabilities may be out of date, a background query is made automatically to get the latest state to keep this property up-to-date. You can simply subscribe to PropertyChanged on this class to be notified of changes.
