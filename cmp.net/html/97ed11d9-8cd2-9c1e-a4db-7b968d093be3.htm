Remarks

You generally don't need to worry whether the client supports any specific keyboard configuration. If you request a configuration that the client does not support then your request will get automatically downgraded to a less capable configuration that the client does support.

The keyboard capabilities are updated whenever a client reconnects. Use SessionConnected to detect when a client reconnects.
