PictureCopyFileToTemp Method Citrix Mobility Pack .NET SDK
Attempts to copy the picture located on the client device to the user's local temporary directory on the Citrix server. Any existing file in the temporary directory with the same name will get overwritten. You can test whether the source file is available using FileExists.
