CameraCanControlThumbnailSize Property Citrix Mobility Pack .NET SDK
Reports whether the client supports the capability to create a thumbnail with a non-default size, i.e. with the ThumbnailType set to one of:
  • Small
  • Medium
  • Large
  • Custom
This only applies to pictures taken with the V2 take picture API: BeginTakePicture2(PictureOptions, TimeSpan, ActionIAsyncResult) or TakePicture2(PictureOptions, TimeSpan).
