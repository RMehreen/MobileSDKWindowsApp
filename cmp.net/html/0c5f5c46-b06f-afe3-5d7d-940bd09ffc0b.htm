VideoCaptureOptions PropertiesCitrix Mobility Pack .NET SDK

The VideoCaptureOptions type exposes the following members.

Properties
  NameDescription
CameraSelection
Specifies the preferred camera to make active on the device to capture the video.
DurationLimitSeconds
The limit in seconds for the length of the video clip. The default value is 0 which specifies that there is no duration limit.
Encoding
The encoding algorithm to use to record the video.
Quality
Defines the video quality to use for the capture video request. Lower qualities will give a smaller file size at the expense of image quality.
