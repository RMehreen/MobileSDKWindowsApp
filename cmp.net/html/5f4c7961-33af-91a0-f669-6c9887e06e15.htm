IgnoreForegroundCheck1

This option when enabled, prevents CMP from checking focus of the associated application it is running in when processing requests or events. By default CMP will return a "not in focus" error if you try to call any mutating APIs when your app does not focus. Similarly CMP will not fire any notification events triggered by the client device in your app if it does not focus.

For most apps it is recommended that you do not enable this option. Leaving it disabled will make you app interact more gracefully with other CMP enabled apps running in the same session.
