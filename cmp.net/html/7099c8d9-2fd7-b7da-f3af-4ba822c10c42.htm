ClientState PropertiesCitrix Mobility Pack .NET SDK

The ClientState type exposes the following members.

Properties
  NameDescription
ConnectionState
The status of the underlying connection from the client to the server.
DeviceFormFactor
The form factor of the client device. This value is only accurate when the remote client is a Citrix client that supports CMP. I.e. Protocol is set to CitrixHdxWithCmp. For other Citrix clients without CMP support (Protocol = CitrixHdx) we make a best guess of the form factor using legacy client information.
DeviceOs
The operating system of the client device. This value is only accurate when the remote client is a Citrix client that supports CMP. I.e. Protocol is set to CitrixHdxWithCmp. For other Citrix clients without CMP support (Protocol = CitrixHdx) we make a best guess of the device OS using legacy client information.
IsCitrixProtocol
Whether or not your app is being accessed via one of the Citrix protocols (regardless of whether it supports CMP).
IsMobileFormFactor
Returns true when DeviceFormFactor is reporting a form factor that indicates a mobile device, i.e. Phone or Tablet.
IsRemoteProtocol
Returns whether the user is accessing the app via a remote client (using either Citrix HDX or Microsoft RDP).
Protocol
Returns the underlying protocol in use to remote the application from the server it is running on, to the client the user is using.
