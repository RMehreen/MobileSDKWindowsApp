Picker PropertiesCitrix Mobility Pack .NET SDK

The Picker type exposes the following members.

Properties
  NameDescription
HasPickerBeenShown
Used to track whether a picker has been shown or not. Once you have shown a picker you cannot show it again - you have to create a new picker instance.
ItemCount
The number of items in this picker.
Items
The picker items that will be displayed to the user.
PickerLayout
Used to set the size and position of the picker control. This layout rect has a default value of (0, 0, 0, 0) which makes the client automatically layout the picker control. Generally you will not need to alter this value.
PickerTitle
Used to set the picker title that is displayed on the picker. This defaults to null so that no title is displayed. If the client does not support displaying a title on a picker then this title will be ignored. You can check whether the client supports displaying the title using CanSetPickerTitle.
SelectedItem
Returns the selected item, or null if no item is selected.
SelectedItemIndex
The index of the selected item. Set this prior to showing the picker to set the default selected item. Once the user has successfully picked an item this will be set to the value of the picked item. If no item is selected this property will return -1.
