ClientState ClassCitrix Mobility Pack .NET SDK
This class reports a range of useful information about the current client the user is using to access your application. You can use this state to alter the appearance or behaviour of the application to best suit the way the application is being accessed.

Remarks
When the user is running your app from a Citrix Receiver that supports the Mobility SDK all the client state gets queried from the Receiver and will be accurate. However if the user is using a Citrix Receiver that does not support the Mobility SDK then not all of the client state will be completely accurate. Use IsCmpAvailable to determine whether the Mobility SDK is support by the currently connected Citrix Receiver.
