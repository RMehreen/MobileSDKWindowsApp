Parameters

zoomPercent
Type: SystemInt16
The viewport zoom percentage. Use 100 to apply the default 1:1 scaling. Use values higher than 100 to zoom in, and lower values to zoom out. You must supply a value of [1..1000], or an exception will be thrown.
Exceptions
ExceptionCondition
ArgumentOutOfRangeExceptionThrown if you supply an invalid zoom percentage.
CmpCallFailedExceptionThrown when the request was successfully issued via the CMP runtime, but the request returned an error.
CmpExceptionThrown when there is any internal error that prevents the request from being issued to the CMP runtime.
NotSupportedExceptionThrown if the current client does not support the capability to set the viewport zoom level. You can call CanSetViewport to check whether this capability is supported.
