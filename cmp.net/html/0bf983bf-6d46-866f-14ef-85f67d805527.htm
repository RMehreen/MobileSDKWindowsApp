The OrientationState type exposes the following members.

Properties
  NameDescription
ApplicationOrientation
This is the orientation the application is in. When no orientation locking is being applied this will always be the same as the device orientation. When locking is being applied then this will be the orientation the locking is holding the application in.
DeviceOrientation
This is the orientation the device is currently being held in, regardless of any orientation locking that has been applied.
IsOrientationLocked
Whether or not any orientation locking is being applied.
