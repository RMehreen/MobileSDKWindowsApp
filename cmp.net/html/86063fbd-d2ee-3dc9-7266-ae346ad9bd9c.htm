The Device type exposes the following members.

Properties
  NameDescription
DeviceOs
Returns the operating system of the client device. If there is no connected client, the client doesn't support CMP, or the client operating system is not known then the Unknown type will be returned.
DeviceType
The device type specifies the form factor of the client device. If there is no connected client, the client doesn't support CMP, or the client type is not known then the Unknown type will be returned.
OsVersion
Returns the operating system version of the client device. If the version could not be obtained, this will return null.
