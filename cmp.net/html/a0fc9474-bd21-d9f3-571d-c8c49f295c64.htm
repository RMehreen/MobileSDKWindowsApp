IsAutomaticRequeryDisabled
Property to enable or disable CommandManager's automatic requery on this command. When automatic requery is enabled any subscription to CanExecuteChanged will hence get added to CommandManager.RequerySuggested as well. When the value is changed all existing handlers are subscribed or unsubscribed from the CommandManager.RequerySuggested as appropriate.
