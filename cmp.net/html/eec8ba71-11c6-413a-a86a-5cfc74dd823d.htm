DisplaySetSupportedOrientations Method Citrix Mobility Pack .NET SDK

This API locks the device to the specified set of orientations.

The typical usage of this API is to lock the device to portrait or landscape. While you can lock a device to one of the four possible orientations, it is recommended that you always lock to pairs of orientations. So if you want to lock to landscape you should lock to landscape left and landscape right. This allows the user to hold the device in either landscape orientation and the app will auto-rotate between the two, without going into portrait.

You can use OrientationLockingV2Supported to check whether the current client supports orientation locking. If it does not support it the operation will throw a NotSupportedException which you will receive when you complete the async operation.
