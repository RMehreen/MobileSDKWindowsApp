LayoutTracker ClassCitrix Mobility Pack .NET SDK
The layout tracker is used to query the form factor of the endpoint the user is running your application from. By default the layout is queried from the Citrix Receiver using the Mobility Pack runtime. However you can supply a custom form factor report if required.

Remarks
Most applications will never need to define a custom form factor report. One case where a custom report is useful is when you want to manually support toggling between different layouts, e.g. using a setting in your application.
