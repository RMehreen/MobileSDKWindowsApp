BeginHidePicker
Initiates an async request to hide the specified picker if you want to hide it without waiting for the user to select an item or hide it themselves.
BeginShowPicker
Initiates an async request to display a picker to the user. You must first create your picker using the CreateNewPicker(IEnumerableString) function to supply as the picker argument. You must create a new Picker instance every time you want to display a picker, you cannot show the same picker instance multiple times.
CreateNewPicker
Creates a new picker control to display locally on the client. You must create a new picker control for each show picker request that you make. If you want to reuse an existing picker instance you can clone it using Clone.
