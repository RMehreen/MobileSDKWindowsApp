AudioClip PropertiesCitrix Mobility Pack .NET SDK

The AudioClip type exposes the following members.

Properties
  NameDescription
ClipFileName
The file name of the audio clip on the client device. It is recommended that you copy this file to the local (Citrix) server before attempting to use it. This is because the client may connected by a slow, wireless network.
Duration
The length of the audio clip.
Encoding
Returns the encoding that was used to capture this audio clip.
FileExists
Tests whether or not the underlying file on the client device that holds the clip exists and is currently accessible. When the client becomes disconnected the file will no longer be accessible.
Quality
The quality used for the capture operation. The quality is a percentage value of: [1..100], where one is the lowest quality and one hundred is the highest. If the quality is unknown then zero is returned.
Size
The file size of the audio clip in bytes.
