Camera ClassCitrix Mobility Pack .NET SDK
This class is used to take pictures using the camera of the client device. The pictures are stored on the local storage of the device. The Picture objects the API returns contain the remote file path of the pictures, and you can use these to copy the pictures to the local Citrix server so your app can use them.

Initiates an async request to remove the specified picture. Pictures are stored on the client device in the private application data store for the Citrix Receiver. TakePicture
Brings up the native camera app on the device so the user can take a picture. This function will block until the user has taken the picture or cancels the operation, or the supplied timeout occurs. Remarks
Take picture requests will usually take a long time to return since you have to wait until the user takes the picture (or the request times out). Also the version 2 take picture API BeginTakePicture2(PictureOptions, TimeSpan, ActionIAsyncResult) may perform additional post processing on the image file before returning. For these reasons you avoid using the synchronous versions of the take picture APIs. If you do call the synchronous versions you should not call them from your UI thread, since your UI will lockup until they return.
Examples
The following sample shows how you can take a picture and copy the picture off the client device to the local Citrix server your app is running on.
Copy
CmpApi cmpApi = new CmpApi();
cmpApi.Camera.BeginTakePicture(PictureImageFormat.Jpeg, TimeSpan.FromSeconds(30), ar =>
{
    try
    {
        var picture = cmpApi.Camera.EndTakePicture(ar);

        // Copies the picture off the client device to the Citrix server your app
        // is running on. You should always copy the picture to Citrix server since
        // accessing the picture over a wireless network to the client device can
        // be slow. Note that we are performing the copy in a worker thread since
        // the operation can take a long time to complete and we don't want the UI
        // thread to block while we are doing this.
        ThreadPool.QueueUserWorkItem(workItem =>
            {
                var tempFileName = picture.CopyFileToTemp();
            });
    }
    catch (CmpException ex)
    {
    }
    catch (NotSupportedException ex)
    {
    }
    catch (TimeoutException ex)
    {
    }
});
