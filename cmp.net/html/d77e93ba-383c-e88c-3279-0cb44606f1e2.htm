DeviceDependentSeparatorHeightKey

This resource key returns a double value for the height in DIUs that will draw horizontal separator lines with a consistent physical pixel size. To implement DPI scaling, WPF uses pixel rounding when its device independent values do not end up producing a value that gives an even number of pixels. This means over a series of separator lines some lines will be rounded up while others get rounded down, giving an inconsistent thickness.

Using this property will remove the inconsistent thickness between separator lines, as well as ensure no lines get rounded down to zero pixels. This key attempts to get a thickness as close to one DIU as possible.

SessionDpiX
Returns the in-session DPI. When running in a XenApp server session the session DPI will always be 96. The session DPI can only vary from 96 when your app is running in a XenDesktop session or when you are logged on directly to the computer (at the console).

Zoom
This property allows you to further adjust the scale of your application in addition to the scaling that gets applied based on the client device form factor and DPI. The default value is 1.0 (100%) for no zoom. Use lower values to zoom out, and higher values to zoom in. Attempts to set the zoom to 0 or below will be ignored.
