Display EventsCitrix Mobility Pack .NET SDK

The Display type exposes the following members.

Events
  NameDescription
DisplayStateChanged
Fired whenever the display state changes. This will get fired initially on a connection/reconnection with the state queried from the capabilities negotiation. It will then get fired again if it ever changes.
OrientationStateChanged
Fired whenever the device orientation, application orientation or orientation locking state changes.
SupportedOrientationsChanged
Fired whenever the supported orientation state changes. The supported orientations are a set of flags that report which orientation(s) a device has been locked to.
ViewportInfoChanged
Fired whenever the client or server viewport information changes.
ViewportOriginChanged
Fired whenever the viewport origin changes.
