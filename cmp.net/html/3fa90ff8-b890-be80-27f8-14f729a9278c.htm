PictureQueryMetaData Method Citrix Mobility Pack .NET SDK
This function queries the video clip meta-data from the client device and parses it into a map of key/value pairs. If you need to use the meta-data, you should call this function as soon as possible after capturing an audio clip to ensure you have access to it. This is because the client can get disconnected at any time, and you will lose access to the meta-data once the client is disconnected.
