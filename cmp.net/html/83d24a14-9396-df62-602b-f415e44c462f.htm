Events
  NameDescription
ItemSelected
Fired when the user selects an item. Once an item has been selected no further events will be triggered for this picker.
PickerClosed
Fired when the user closes the picker without selecting an item. Once a picker has been closed, no further events will be triggered for this picker.
PickerError
Fired when there is any error for this picker. Once an error has occurred, no further events will be triggered for this picker.
