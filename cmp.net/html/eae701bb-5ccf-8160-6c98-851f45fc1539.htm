Input EventsCitrix Mobility Pack .NET SDK

The Input type exposes the following members.

Events
  NameDescription
ButtonPressed
This event is fired whenever the user presses a button on the client device that has been configured to be redirected to the server, i.e. your application.
ButtonRedirectionChanged
This event is fired whenever the redirection state of a button changes. Note you should also subscribe to SessionConnected. When a session reconnects the redirection status of all buttons will always be reset to ClientDevice.
KeyboardStateChanged
This event is fired whenever a change in the keyboard state is detected.
TouchInputModeChanged
Fired whenever the touch input mode changes.
