Picture PropertiesCitrix Mobility Pack .NET SDK

The Picture type exposes the following members.

Properties
  NameDescription
Camera
Returns the camera that was used to take the picture. This is only set for V2 take picture requests. V1 take picture requests will always return Default, regardless of which camera was used.
FileExists
Tests whether or not the underlying file on the client device that holds the picture exists and is currently accessible. When the client becomes disconnected the file will no longer be accessible.
FileName
The full path and file name of the picture. This file is located on the client device itself, so it is recommended that you first copy the file to a local location on the Citrix server before you attempt to use it. This will make repeated access faster, and ensures you can still access the picture if the client becomes disconnected. You can use any standard file IO API to copy the file from this source location (as long as the client is still connected).
Format
The image format of the picture.
IsV2Picture
Whether or not this picture was taken using the version 2 take picture API. The version 2 API supports additional features to control the size and quality of the image. It also returns extra information, including EXIF tag meta-data and a thumb- nail view of the picture.
Quality
The quality used for the capture operation. The quality is a percentage value of: [0..100]. A value of 0 means the default quality of the device was used to take the picture. One is the lowest quality and one hundred is the highest quality.
Resolution
Returns the resolution of the picture. This resolution is only set for V2 take picture requests. V1 requests will always return (0, 0).
Size
The file size of the picture in bytes.
ThumbnailExists
Returns whether or not there is a thumbnail image available for this picture on the client device.
ThumbnailFileName
Thumbnail file name. This is the name of an image file on the client device that contains a reduced thumbnail view of the image. A thumbnail is only available for pictures that are taken using the V2 take picture API.
ThumbnailType
The type of thumbnail that is available for this picture. Thumbnails can only be generated by V2 take picture requests. V1 pictures will always return None.
