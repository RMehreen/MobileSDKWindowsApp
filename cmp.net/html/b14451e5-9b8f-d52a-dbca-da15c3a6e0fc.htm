InputIsKeyboardTypeSupported Method Citrix Mobility Pack .NET SDK
Checks the supplied keyboard type against the current keyboard capabilities to determine whether the current connected client supports showing that type of keyboard. The unknown and physical keyboard types will always return false since we can only ever request an onscreen keyboard type to be shown.
