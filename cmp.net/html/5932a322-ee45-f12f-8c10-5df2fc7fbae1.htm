ClientControlsBeginEnable Method Citrix Mobility Pack .NET SDK

Initiates an async request to enable the on-screen Citrix Receiver client controls if you have previously hidden them using Disable or BeginDisable(ActionIAsyncResult). To check the result of the request, call EndEnable(IAsyncResult).

You can test whether the current client supports controlling the on-screen controls using CanControl.

This API supports fallback. This means if you disable or enable the controls when the client does not support this then you will not see an error and your desired state will be cached. If the user then connects via a different client that does support this capability then the cached state will be applied and the client controls will get disabled or enabled as appropriate.
