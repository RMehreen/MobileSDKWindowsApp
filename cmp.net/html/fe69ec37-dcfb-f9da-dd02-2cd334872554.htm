PictureOptions PropertiesCitrix Mobility Pack .NET SDK

The PictureOptions type exposes the following members.

Properties
  NameDescription
CameraSelection
Specifies the preferred camera to make active on the device to take the picture.
DesiredHeight
The desired height of the image in pixels. Leave this value 0 to use the default of the device.
DesiredWidth
The desired width of the image in pixels. Leave this value 0 to use the default of the device.
Encoding
Specifies the preferred image encoding you want to receive the image in.
Quality
Defines the picture quality to use for the take picture request. Lower qualities will give a smaller file size at the expense of image quality.
ThumbnailHeight
The desired height of the thumbnail in pixels. This is only used when you set ThumbnailType to Custom
ThumbnailType
Set the thumbail type to control both the size of the associated thumbnail or whether any thumbnail is created. If you don't want to use any of the preset sizes you can supply the custom type, and then set ThumbnailWidth and ThumbnailHeight to control the pixel size of the thumbnail.
ThumbnailWidth
The desired width of the thumbnail in pixels. This is only used when you set ThumbnailType to Custom
