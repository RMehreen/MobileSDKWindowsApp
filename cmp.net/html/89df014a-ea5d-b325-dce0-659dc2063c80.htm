DisplayState PropertiesCitrix Mobility Pack .NET SDK

The DisplayState type exposes the following members.

Properties
  NameDescription
ColorDepth
The color depth of the client device, or null if it could not be queried (either the client doesn't support reporting it or there was an error querying it).
Dpi
The DPI of the client device, or null if it could not be queried (either the client doesn't support reporting it or there was an error querying it).
Resolution

The resolution of the client device, or null if it could not be queried (either the client doesn't support reporting it or there was an error querying it).

This resolution specifies the full screen resolution of the client device. The actual resolution available to display the session maybe less than this due to tool- bars. To get the available resolution for displaying the session, you need to query the client viewport instead.

ScreenSize
The screen size of the client device, or null if it could not be queried (either the client doesn't support reporting it or there was an error querying it).
